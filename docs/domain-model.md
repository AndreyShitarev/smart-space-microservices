# Проектирование доменной области (этап 1)
## Бизнес-концепция и сценарии

**Продукт:** Samrt Space - платформа для бронирования рабочих мест и переговорок в коворкинге.
**Цель:** Автоматизировать процесс поиска свободного пространства и его бронирования, исключив двойные брони

## Основные сценарии (Use Case):

1. **Поиск и просмотр:** пользователь фильтрует переговорки по вместимости и оборудованию в них и смотрит расписание
2. **Бронирование:** пользователь выбирает слот времени и оплачивает или подтверждает свою бронь
3. **Отмена:** пользователь отменяет бронь в соответствии с правилами возврата
4. **Администрирование:** админ добавляет новые переговорки и меняет их статус (например, "На ремонте")
