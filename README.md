# Smart Space: Микросервисная система бронирования коворкингов

## О проекте
Система позволяет пользователям находить и бронировать переговорные комнаты в коворкинге. Арихитектура построена на микросервисах для обеспечаения масштабируемости и независисимого развертывания компонентов.

## Структкра документации 

* Этап 1 - Доменная модель и бизнес-правила (docs/domain-model.md)
* Этап 2 - API контракты и взаимодействие сервисов (api/api-conracts.md)
